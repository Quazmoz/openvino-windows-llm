# OpenVINO Windows LLM 0.6.2

This reliability release binds every publishable artifact set to its clean source commit.

- Cryptographically binds release checksums, summaries, manifests, and the built model-library asset to the tag commit.
- Ranks exact-model local benchmarks ahead of bundled certification only when requested and actual direct devices agree.
- Recommends verified GPU execution for Qwen 2.5 3B FP16 while retaining manual NPU selection.
- Bounds race-safe Windows converter process-tree cancellation and permits immediate retry after cleanup.
- Expands CI syntax and targeted regression coverage for release, identity, recommendation, and cancellation gates.
- Uses precise, machine-scoped NPU compatibility wording instead of unsupported universal claims.

Artifacts produced with `-Unsigned` are not Authenticode-signed. This release does not resolve the
certificate-backed signing and clean-machine signed upgrade/uninstall work tracked in issue #15.
